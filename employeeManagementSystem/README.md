# Employee Management System

This is a simple Employee Management System project developed using Java, Spring Core, Spring JDBC, Maven, and MySQL.

The project performs CRUD operations:

* Add Employee
* Search Employee
* Update Employee
* Delete Employee

## Technologies Used

* Java
* Spring Core
* Spring JDBC
* Maven
* MySQL

## Project Structure

```text
Controller → Service → Repository → Database
```

## Database Table

```sql
create table emp1(
   eno int primary key,
   ename varchar(30),
   esal float,
   eaddr varchar(30)
);
```

## Features

* Layered Architecture
* Dependency Injection
* JDBC Connectivity
* Maven Project
* CRUD Operations
